# livecdn.github.io
## Simple CDN you can use

### DateTime Picker

Usages:

```sh
<link href="//livecdn.github.io/cdn/date.css" rel="stylesheet">
<script src="//livecdn.github.io/cdn/date.js"></script>
$('.date').datetimepicker({
    timepicker:false,
    format:'Y-m-d',
    scrollMonth: false,
    scrollInput: false
});
```
